Local. Offline. Simple.

Convert PowerPoint files to PDF on your desktop.

SlideDrop is a small desktop utility for Windows and macOS that converts .ppt and .pptx files to PDF using LibreOffice on your computer.

Download Requirements

Built for batch conversion

Add files or folders, review the queue, and convert everything in one local batch. Failed files do not stop the rest of the queue.

No cloud upload

SlideDrop calls LibreOffice in headless mode through a local subprocess. Your PowerPoint files stay on your machine.

Practical output behavior

Folder scans write PDFs into a local pdf folder. Individually selected files write PDFs beside their source files.

macOS

macOS packaging is prepared, but public builds are not published yet.

Must be built on macOS with build-macos.sh.

Requirements

  • LibreOffice installed locally.
  • Windows default path: C:\Program Files\LibreOffice\program\soffice.exe
  • macOS default path: /Applications/LibreOffice.app/Contents/MacOS/soffice

PDF preserves static slide visuals, but not PowerPoint animations, transitions, audio, video, or interactive media behavior.
